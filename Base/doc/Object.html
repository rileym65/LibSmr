char* AsCharArray()

Function: Returns character array of object's type

Notes:

  • This returns a pointer to the object type within the object. The memory for this is owned by the object and therefore free() should not be used on the return value.

char* Equals(Object* b)

Arguments:
        b - Reference to an object to compare

Function: Returns true if 'b' is the same object

Notes:

  • This returns true if the object pointed to by 'b' is the same as the current object.

char* ToCharArray()

Function: Returns character array of object's type

Notes:

  • This function returns a new memory object with a copy of the object's type. The return value of this function is created by malloc() and therefore you should use free() once this value is no longer needed.

char* Type()

Function: Returns character array of object's type

Notes:

  • This returns a pointer to the object type within the object. The memory for this is owned by the object and therefore free() should not be used on the return value.
